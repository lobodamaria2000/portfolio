# Maria Loboda Portfolio

Static portfolio website for GitHub Pages.

## Files

- `index.html` - page content
- `styles.css` - visual design and responsive layout
- `script.js` - theme toggle and mobile menu
- `assets/` - local visual assets

## GitHub Pages

1. Create a repository, for example `portfolio`.
2. Push these files to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Set source to `Deploy from a branch`, branch `main`, folder `/root`.
5. Save and wait for GitHub to publish the site.

For a personal domain later, add it in the same Pages settings and enable HTTPS.
